Easy Deployment

Get your AdvisorX CRM up and running in minutes

Deployment Options

1

Local Testing

Test the application locally before deployment: 

cd /Users/test/startups/advisorymanagement/frontend/out
npx serve -p 3000

Then visit: http://localhost:3000

2

Vercel Deployment

Deploy to Vercel for free hosting:

  1. Go to https://vercel.com/new
  2. Import the static files from: 
    /Users/test/startups/advisorymanagement/frontend/out
  3. Follow the prompts to complete deployment
3

Alternative Hosting

The application can be deployed to any static hosting service:

  • Netlify (drag and drop the out folder)
  • GitHub Pages
  • AWS S3
  • Firebase Hosting
  • Any traditional web server
